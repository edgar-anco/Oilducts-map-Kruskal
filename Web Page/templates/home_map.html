Bienvenido

A continuación, visualizarás el mapa de las interconexiones de petróleo y gas natural de EEUU. Se utilizó el algoritmo de Kruskal para encontrar el árbol de expansión mínima que permita conectar a todos los nodos y así proponer un sistema optimizado para la red de petróleo y gas natural de EEUU.

Se utilizó una BD abierta del gobierno estadounidense para obtener el dataset de nodos de puntos de interconexión.
